This is a simple login and signup app I built using React, Node.js, Express, and Firebase.

Tech Used
React (frontend)
Node.js + Express (backend)
Firebase (authentication & database)
Features
User signup
User login
Password validation (strong password rules)
Firebase authentication
User data stored in database
API keys hidden using .env
Security
Used environment variables to hide API keys
Firebase handles authentication securely
Basic input validation added
How It Works
User opens the app → sees login page
User can sign up if they don’t have an account
After signup → redirected to login
After login → user is welcomed (example: “Hi username”)
Run the Project
npm install
npm start

For frontend:

cd client
npm install
npm start
