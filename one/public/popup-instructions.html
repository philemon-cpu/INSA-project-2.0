🔓 Enable Popups for Google Sign-In

To use Google Sign-In, you need to allow popups for this website. This is a security feature in modern browsers that you can easily configure.

🌐Chrome/Edge:

1. Look for the popup blocker icon (🚫) in the address bar
2. Click the icon and select "Always allow popups from this site"
3. Refresh the page and try Google Sign-In again

🦊Firefox:

1. Click the popup blocker icon in the address bar
2. Select "Allow popups for this site"
3. Refresh the page and try Google Sign-In again

🧭Safari:

1. Go to Safari > Preferences > Websites
2. Select "Pop-up Windows" from the left menu
3. Find this website and set it to "Allow"
4. Refresh the page and try Google Sign-In again

⚙️Alternative: Redirect Method

If you prefer not to enable popups, the app will automatically use a redirect method that doesn't require popups. This will temporarily redirect you to Google and back.
